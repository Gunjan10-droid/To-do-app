# ✅ TickTask - Your Personal Todo Manager

**TickTask** is a simple and elegant Todo list app built with **React**. It allows you to manage your daily tasks, mark them as completed, edit or delete them, and stores everything locally using **localStorage**, so your data persists even after refreshing the browser.

---

## 🚀 Features

- 📝 Add new todos
- ✅ Mark tasks as completed
- 🛠️ Edit existing tasks
- 🗑️ Delete tasks
- 👀 Toggle visibility of completed tasks
- 💾 Saves data in localStorage
- 📱 Responsive design using Tailwind CSS

---

## 🛠️ Tech Stack

- **React** (Functional Components + Hooks)
- **Tailwind CSS** (for styling)
- **UUID** (to generate unique IDs for todos)
- **React Icons** (for icons)
